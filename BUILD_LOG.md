### Design Decisions

**Color Scheme:**
- Primary: Purple (#8B5CF6) - Trust, premium
- Secondary: Teal (#14B8A6) - Calm, organized
- Accent: Pink (gradient) - Energy, friendly
- Background: White with subtle gradients

**Typography:**
- Font: Inter (clean, modern, readable)
- Headings: Bold, large, gradient text
- Body: Gray-700 for readability

**Voice/Tone:**
- Friendly but professional
- Empathetic (understands pain)
- Aspirational (better life possible)
- Authentic (real founder story)

### Technical Decisions

**Why Next.js 14?**
- Modern React framework
- App Router for clean structure
- Built-in optimization
- Easy Vercel deployment
- Same stack for landing + app (consistency)

**Why Tailwind CSS?**
- Rapid development
- Mobile-first by default
- No CSS file management
- Easy to iterate
- Production-ready

**Why Vercel?**
- Free tier sufficient for Phase 0
- Automatic deployments from Git
- Built-in analytics
- Edge network (fast globally)
- Easy custom domain setup
